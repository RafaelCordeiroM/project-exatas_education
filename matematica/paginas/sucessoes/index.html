Método para extração da raiz quadrada

Exemplo ilustrativo do método da raiz quadrada.

Dado um número positivo qualquer c {\displaystyle c} , desejamos encontrar um número x {\displaystyle x} positivo tal que x 2 = c {\displaystyle x^{2}=c} . Suponhamos, agora, que nos é conhecida apenas uma aproximação x ~ > 0 {\displaystyle {\tilde {x}}>0} para x {\displaystyle x} . Notemos que:

x ~ c x ~ = c {\displaystyle {\tilde {x}}{\frac {c}{\tilde {x}}}=c}

e, observamos que:

  1. c {\displaystyle {\sqrt {c}}} é um valor entre x ~ {\displaystyle {\tilde {x}}} e c x ~ {\displaystyle {\frac {c}{\tilde {x}}}} ;
  2. se a aproximação x ~ {\displaystyle {\tilde {x}}} aumenta de valor, então o fator c x ~ {\displaystyle {\frac {c}{\tilde {x}}}} diminui e vice-versa;
  3. x {\displaystyle x} é solução de x 2 = c {\displaystyle x^{2}=c} , se x = c x {\displaystyle x={\frac {c}{x}}} .

Destas observações, vemos que uma melhor aproximação para c {\displaystyle {\sqrt {c}}} pode ser obtida tomando a média aritmética entre x ~ {\displaystyle {\tilde {x}}} e c x ~ {\displaystyle {\frac {c}{\tilde {x}}}} , i.e.:

a 1 = 1 2 ( x ~ + c x ~ ) {\displaystyle a_{1}={\frac {1}{2}}({\tilde {x}}+{\frac {c}{\tilde {x}}})} .

Agora, a 1 {\displaystyle a_{1}} é uma nova aproximação de x {\displaystyle x} e, repetindo o argumento acima, temos que a média:

a 2 = 1 2 ( a 1 + c a 1 ) {\displaystyle a_{2}={\frac {1}{2}}(a_{1}+{\frac {c}{a_{1}}})}

é uma aproximação para x {\displaystyle x} ainda melhor que a 1 {\displaystyle a_{1}} .

Seja, então, ( a n ) n {\displaystyle (a_{n})_{n}} a sequência definida recursivamente por:

a 1 = x ~ , a n = 1 2 ( a n − 1 + c a n − 1 ) , n = 2 , 3 , . . . {\displaystyle a_{1}={\tilde {x}},\quad a_{n}={\frac {1}{2}}(a_{n-1}+{\frac {c}{a_{n-1}}}),n=2,3,...} .

Podemos mostrar que a n {\displaystyle a_{n}} converge para c {\displaystyle {\sqrt {c}}} . Esta sequência tem origem na Mesopotâmia (séc. XVIII a.C.) e é talvez o método mais eficiente para extração da raiz quadrada.[6]
